# Sales & Cunstomer Analytivs Dashboard
## Project Overview
This project analyzes sales and customer data to generate business insights and build an interactive dashboard.

## Objectives
- Identify top-performing products
- Analyze sales trends over time
- Evaluate regional performance
- Understand customer behavior
- Provide business recommendations

## Tools & Technologies
- Python (pandas, matplotlib)
- SQL
- Power BI
- Excel (optional)

## Dataset
The dataset contains:
- Orders
- Customers
- Products
- Regions
- Sales
 
## Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized column names
- Converted date columns
- Created new features (year, month)

##  Key Insights
- A small number of products generate most of the revenue
- Sales show seasonal trends
- Some regions underperform compared to others
- Certain products generate losses
